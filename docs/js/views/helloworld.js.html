"el" is the top-level HTML element of the view, within which the view's content will be rendered. By default, it is an anonymous HTML div element. Alternatively, the view instantiator can set the value of el, or the view can set its own el (generally avoided, since that makes the view too tightly coupled to its use). Here we use the default "div" el.
